# linux

### 文件

##### 文件隐藏权限

- chattr         重要参数 i a
- lsattr         查看隐藏权限

##### 文件默认创建权限

- 文件 默认666    目录默认  777
- umask   去除的权限

##### 文件特殊权限

- s权限仅对二进制可执性文件有效

##### 查看文件类型

- file

##### 查找文件

- which  只找PATH路径
- whereis只找特点的路径
- locate找数据库里记载的  若未更新数据库 则可以updatedb
- find 找磁盘 很慢
